Feature: Todo List

Purpose

Users can manage a list of todos.

User Flow

1. User enters todo text
2. User clicks add button
3. Todo item appears in the list
4. User can toggle todo completion
5. User can delete a todo
6. Completed todos are visually marked

Behavior

- empty todo cannot be added
- todos are displayed in a list
- completed todo should be styled differently
- deleting removes todo from list
